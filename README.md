# OnlyArial's Tools

A collection of Minecraft UI tools inspired by classic community utilities.

## Included Tools

### 5-Bit Font Converter

Convert normal text into a compact 5x5 UI-style font for Minecraft menus, items, ranks, and interfaces.

* Supports A-Z and 0-9
* One-click copy output
* Clean modern UI
* Self-hostable

### Tag Tool

Create pixel-art Minecraft rank tags and UI labels with gradients, shadows, rounded corners, and customizable padding.

* True 5x5 pixel font rendering
* Gradient backgrounds & text
* Pixel-rounded corners
* PNG export & clipboard copy
* Fully browser-based
